To get a bra that fits properly, it is vital to measure your bra size. This free calculator will tell you how to measure your breast size.

[https://www.calculator.io/bra-size-calculator/](https://www.calculator.io/bra-size-calculator/)

== Usage ==

Add the Bra Size Calculator shortcode to your page, post or sidebar:

`[ci_bra_size_calculator]`

Add the following code to your template where you would like the Bra Size Calculator to appear:

`<?php display_ci_bra_size_calculator(); ?>`
